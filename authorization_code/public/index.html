MY DUNGEON CRAWLER

I am currently working on a dungeon crawler game for my enjoyment and experience. The game is constantly being worked on and it is my hope to one day build this up into a full, fun, and extensive game that many people could find enjoyment in. I am writing the game in C# using Unity 3D. The main goal for my game is to create a system where the player creates a party of 4 characters and goes out into a dungeon with random encounters, levels up, gains items, and challenges bosses. If you wish to check out the progress on my game so far please download one of the links below. Please keep in mind that I'm still learning a lot about Unity so be aware that there are a lot of bugs in the game. I try and push builds of my game to the server whenever I feel that I have reached a notable checkpoint in my development, so if you want to see how the game is progressing come back and check it out every once in a while.

SPOTIFY SAFE PLAYLISTS

This is a Spotify program that creates “safe” playlists based off the user saved music. The definition of “safe” in the case of this program is songs that are popular enough to be considered mainstream. The program works by grabbing all of a users saved songs from Spotify and then creating a playlist of “safe” music in the users library.
